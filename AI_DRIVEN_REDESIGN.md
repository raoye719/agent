# 动态规划系统 - AI驱动的重新设计

## 核心理念转变

### 旧设计（结构化输入）
```
用户 → 填充JSON表单 → 系统处理 → 输出计划
❌ 繁琐、不自然、没有发挥AI优势
```

### 新设计（自然语言驱动）
```
用户 → 自然语言描述周计划 → AI提取精炼 → AI自主分配日计划 → 动态调整
✅ 自然、智能、充分发挥AI能力
```

---

## 新的工作流程

### 第一步：用户输入周计划（自然语言）

**用户输入示例**：
```
我这周的考研计划是：
- 数学要把泰勒公式和积分应用搞透彻，计划花15小时
- 英语要完成20篇阅读理解，每天背200个单词，计划花12小时
- 政治要完成马克思主义基本原理第一章，计划花8小时
- 专业课要做完2020年的真题，计划花10小时
这周一共有50小时的学习时间
```

### 第二步：AI处理周计划

**AI做的事**：
1. 理解用户的周计划描述
2. 提取关键信息（科目、目标、时间）
3. 精炼和优化计划
4. **自主分配到每一天**（考虑难度、时间等）
5. 生成结构化的周计划

**AI输出示例**：
```
✅ 我已经理解了你的周计划，这是我的分配方案：

📊 周计划总览
- 总学习时间：50小时
- 科目数：4个
- 计划天数：7天

📅 每日分配
- 周一：数学(3h) + 英语(2h) + 政治(1h) = 6h
- 周二：数学(2h) + 英语(2h) + 专业课(2h) = 6h
- 周三：数学(3h) + 英语(2h) + 政治(1h) = 6h
- 周四：英语(2h) + 专业课(3h) + 政治(1h) = 6h
- 周五：数学(2h) + 英语(2h) + 专业课(2h) = 6h
- 周六：数学(2h) + 英语(2h) + 政治(2h) = 6h
- 周日：复习总结(4h) = 4h

你对这个分配满意吗？如果有调整需求，告诉我！
```

### 第三步：用户可以调整（自然语言）

```

```

```
- 周一：数学(3h) + 英语(2h) + 政治(2h) = 7h

新的周计划已生成，明天早上我会根据这个计划给你安排今日任务。
```

## 新的API设计

### 1. 创建周计划（自然语言输入）

**POST** `/api/ai/plan/weekly/create-from-text`

**请求**：
```json
{
  "weekDescription": "我这周的考研计划是：数学要把泰勒公式和积分应用搞透彻，计划花15小时。英语要完成20篇阅读理解，每天背200个单词，计划花12小时。政治要完成马克思主义基本原理第一章，计划花8小时。专业课要做完2020年的真题，计划花10小时。这周一共有50小时的学习时间。"
}
```

**响应**：
```json
{
  "success": true,
  "message": "✅ 我已经理解了你的周计划，这是我的分配方案：...",
  "weeklyPlan": {
    "weekPlanId": "week-001",
    "weekName": "第一周 2026.3.10-2026.3.16",
    "dailyPlans": [...]
  },
  "needsConfirmation": true
}
```

### 2. 调整周计划（自然语言调整）

**POST** `/api/ai/plan/weekly/adjust`

**请求**：
```json
{
  "weekPlanId": "week-001",
  "adjustmentRequest": "我觉得周一的政治太少了，改成2小时吧。周日的复习时间能不能分散到其他天？"
}
```

**响应**：
```json
{
  "success": true,
  "message": "✅ 已调整！新的周计划已生成...",
  "weeklyPlan": {...}
}
```

### 3. 早上流程（自动执行）

**GET** `/api/ai/plan/daily/morning-auto`

**参数**：
```
weekPlanId=week-001
date=2026-03-10
```

**响应**：
```json
{
  "success": true,
  "message": "🌅 早上好！我检查了周计划和昨天进度，这是今天的计划预览...",
  "dailyPlan": {...},
  "needsAdjustment": false
}
```

### 4. 调整今日计划（自然语言调整）

```json
  "weekPlanId": "week-001",
  "date": "2026-03-10",
  "adjustmentRequest": "我今天上午有课，能不能把数学改到晚上？"
}
```

**响应**：
```json

  "success": true,
```

### 5. 晚上流程（自动执行）

**参数**：

## 核心优势

1. **自然交互**：用户用自然语言交互，无需填表
2. **AI智能**：AI自主分配、调整、分析
3. **灵活适应**：根据实际情况动态调整
4. **持续改进**：每天学习，每周总结，不断优化
5. **充分发挥AI能力**：不是简单的数据存储，而是真正的智能规划
